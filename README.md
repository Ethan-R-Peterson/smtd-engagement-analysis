# SMTD Engagement Analysis
## Overview
This project analyzes student engagement with University of Michigan School of Music, Theatre, and Dance (SMTD) programs and experiences. We used U-M's EXCEL Lab participation data to explore patterns in who engages, how students engage over time, and which experiences drive engagement within SMTD.

The goal is to better understand:

- Engagement trends across majors, class years, and experience types

- Student engagement pathways (first contact → repeat participation → retention)

- Differences in engagement across terms and academic years
